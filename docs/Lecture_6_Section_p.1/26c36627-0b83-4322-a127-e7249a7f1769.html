  1. Bootstrap Cards (карточки)

Как сделать карточку с вкладками или кнопками

Простая карточка

Для того чтобы сделать простую карточку:

  1. Создаем Action
  2. Присваиваем ей роль Section
  3. В родительском элементе устанавливаем Section Style = Card

Карточка с вкладками

Алгоритм Диаграмма Дерево

Для того чтобы сделать карточку с дочерними контент-блоками в виде вкладок:

  1. Создаем Action
  2. Связываем с родительским элементом
  3. В родительском элементе устанавливаем Section Style = Card
  4. Присваиваем созданной Action роль = Section, Section Style = Card Tab
  5. Создаем два (или несколько) дочерних по отношению к карточке элемента
  6. Устанавливаем им роль = Section

Генерируем сайт и получаем карточку с вкладками.
