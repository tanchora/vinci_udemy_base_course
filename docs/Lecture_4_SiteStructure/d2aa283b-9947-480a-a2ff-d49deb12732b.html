Связывание элементов модели

В диаграмме В дереве элементов В поле Linked Elements

Для создания логической структуры модели между элементами должны быть установлены связи, определяющие: какой из элементов является родительским, а какой дочерним.

В диаграмме элементов для этого существует специальный инструмент, который так и называется Child

Для того чтобы установить связь:

  1. Выделяем инструмент Child в палитре.
  2. Наводим мышь на родительский элемент. Появляется иконка розетки. Щелкаем по родительскому элементу.
  3. Наводим мышь на дочерний элемент, снова появляется иконка розетки и мы щелкаем по дочернему элементу.

Если вы неправильно установили связь, то ее можно перетащить к другому элементу или удалить.

У каждой Action во внутренней структуре есть папка Linked Elements. В этой папке хранятся ссылки на дочерние Action.

Установить связь с родительской Action можно перетащив в родительскую папку Linked Elements дочерний элемент модели.

! Удалять элементы из папки Linked Elements нельзя - они полностью удалятся из модели!!!

После того как связи в дереве элементов или на диаграмме будут установлены, дочерние Action отобразится в поле Linked Elements родительской Action.

Для того чтобы установить связь с элементом любой модели, находящейся в рамках того же проекта, используется кнопка с тремя точками. Чтобы удалить связи - кнопка с красным крестом.
