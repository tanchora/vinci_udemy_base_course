В диаграмме

Для создания новых элементов в диаграмме есть специальная палитра, она расположена у правого края окна диаграммы и содержит те элементы модели, которые могут быть созданы в этой репрезентации.

Мы можем выбрать один из трёх способов создания элемента:

  1. Выделить нужный элемент в палитре, затем переместить мышку в область диаграммы и щелкнуть в пустом пространстве.
  2. Схватить нужный элемент мышкой и перетащить его в пустое пространство диаграммы.
  3. Навести мышь на холст диаграммы (пустое пространство) и дождаться пока появится всплывающая палитра инструментов. Навести мышь на нужный элемент и щелкнуть по нему левой кнопкой.

В дереве элементов

В дереве элементов все новые объекты создаются из контекстного меню. Пункты, доступные для выбора зависят от того, из какого элемента модели мы вызываем контекстное меню.

У каждой Action во внутренней структуре есть папка Elements. В этой папке хранятся дочерние Action. Вызвав контекстное меню щелчком на папке Elements и выбрав пункт New Child мы увидим доступные для создания элементы. Выберем пункт Action

Мы можем создать новую Action из контекстного меню родительской Action. Для этого нам потребуется выбрать пункт меню New Child -> Elements -> Action
