Другие виды динамического контента

Формы и календари - это далеко не единственные виды содерания, которое может быть встроено в страницы сайта с помощью контент - элемента Value.

Вы можете встраивать видео, как расположенное локально (что имеет смысл при публикации сайта для внутреннего пользования в локальной сети) или размещенное на ю-тубе.

Так же можно встраивать яндекс или гугл карты, системы поиска, таблицы, документы, презентации и многое другое.

Ниже приведены примеры встраивания разнообразного контента.

Итак, контент-элемент value предоставляет нам возможность встраивания всего, что может быть закодировано и встроено в страницу: html - кода, java script кода и многого другого, что не представляется возможным рассмотреть в рамках данного курса.
