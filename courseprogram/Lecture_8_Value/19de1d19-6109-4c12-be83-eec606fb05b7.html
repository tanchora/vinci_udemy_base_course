    1. Календари

    Яндекс календарь

    Создание яндекс календаря принципиально ничем не отличается от создания гугл календаря.

    Переходим по ссылке https://calendar.yandex.ru

    Воспользуемся ранее созданной учетной записью. В открывшемся календаре увидим, что по умолчанию создался календнарь “Мои события”, чуть ниже находится плюсик, который позволяет создать новый календарь.

    После создания календаря необходимо перейти в его настройки (справа от названия календаря) и установить видимость событий - нажать на переключатель “видят все”.

    Затем перейти на вкладку “Доступ” и скопировать содержимое поля “Код для вставки календаря на сайт”.

    Создать дочерний контент-элемент Value для той Action в которой будет размещен календарь. И вставить код в соответствующее поле.

    Посмотрим на получившийся результат и отметим некоторые моменты, которые портят впечатление от Яндекс - календаря:

    • встроенный календарь не во всех браузерах отражается корректно;
    • нет возможности прикреплять вложения к событию;
    • нет возможности настраивать внешний вид фрейма для встраивания;
    • нет возможности организовать бронирование определенного времени.

    Тем не менее - при желании вы можете пользоваться именно этим сервисом.
