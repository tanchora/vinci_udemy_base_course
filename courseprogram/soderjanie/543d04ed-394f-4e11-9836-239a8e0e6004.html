Создание сайта из репозитория

После того как мы загрузили файлы в репозиторий необходимо установить настройки GitHub pages. Для этого перейдем на вкладку Settings и переходим в настройки репозитория.

Открывается страница с настройками репозитория. Проматываем ее до раздела GitHub pages и выбираем: откуда будет публиковаться сайт: из master branch (главной ветки), из специальной папки в главной ветке или ниоткуда (по умолчанию выбрано None - то есть ниоткуда).

Мы выберем master branch, гитхаб покажет нам предупреждение о том, что настройки изменились. И после этого в разделе GitHub pages появится адрес нашего сайта.

Ура! сайт опубликован!
